 Another quick fix you can try is to repair your game files. If any files are missing or corrupted, the Origin client will add or replace the problematic files for you. Here’s how:

1. Launch Origin, and click**My Game Library** .
2. Right-click Battlefield 2042, then click**Repair Game** .
3. This process may take a while. Restart Origin and run Battlefield 2042 after the repair is complete.

## Fix 3: Install the latest Windows updates

## Fix 4: Force-run DirectX 12

 Per the system requirement, Battlefield 2042 requires DirectX version 12, and the game should run DirectX 12 by default. But we all have different setups on our PC, and it’s possible that Battlefield 2042 isn’t running the right version of DirectX on your PC. You can manually enable DirectX12 and force the game to use it. Here’s how:

1. Press the**Windows key** and**E** on your keyboard to open the file explorer.
2. Navigate to**C:\\Users\\YOUR USERNAME\\Documents\\Battlefield 2042\\** .
3. Find the document named**PROFSAVE\_profile** . Make a copy of this document before tweaking anything, so you’re able to revert the changes in case you need to.
4. Right-click the document and open it with**notepad** .
5. Find the line**GstRender.Dx12Enabled 0** , and**change the value 0 to 1** .
6. Save the changes and launch Battlefield 2042 to test the issue.

 If force-running DirectX 12 for Battlefield 2042 doesn’t help, move on to the next fix.
