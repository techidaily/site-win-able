1. Right-click on Fallout 4 and click **Properties** .
2. In the Compatibility tab, click**Run this program in compatibility mode for** . Then choose the system in the drop-down menu.![](https://images.drivereasy.com/wp-content/uploads/2019/08/com.jpg)

### Fix 3: Edit configuration files

 To fix the black screen issue, you can edit the files to fix it. This method has been proved by many users. Follow the steps below.

1. Navigate to the C:\\Users\\username\\Documents\\mygames\\fallout4.
2. Open**fallout4prefs.ini.** file.
3. Find these variables and change them into these given values.  
 **bMaximizeWindow=0**  
 **bBorderless=1**  
 **bFull Screen=0**  
 **iSize H=1080**  
 **iSize W=1920**
4. Save the changes and run the game.

1. Click the**Play** button in the Steam to run the game.
2. Click**Options** .
