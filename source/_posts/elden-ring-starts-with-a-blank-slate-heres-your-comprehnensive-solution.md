2. Select the**Compatibility** tab. Under**Compatibility mode** section,**check** the box before**Run this program in compatibility mode for** . If you‚Äôre on Windows 10, set it to**Windows 8** . If you‚Äôre on Windows 8, set it to Windows 7\. Click**OK** to finish.
